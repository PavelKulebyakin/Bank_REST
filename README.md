# Система Управления Банковскими Картами

Система предназначена для управления банковскими картами, включая их создание, блокировку, удаление, а также переводы 
между картами и просмотр балансов. Сервис разделяет права доступа для администраторов и обычных пользователей и 
предоставляет REST API с полной документацией.

---

## Возможности

### Администратор
- Создание, блокировка, активация и удаление карт.
- Управление пользователями (CRUD).
- Просмотр всех карт в системе.

### Пользователь
- Просмотр своих карт с поддержкой поиска и пагинации.
- Запрос на блокировку карты.
- Переводы между собственными картами.
- Просмотр баланса карт.

Документация доступна в формате OpenAPI: [`docs/openapi.yaml`](docs/openapi.yaml)

---

## Конфигурация
- JWT секрет передается через глобальную переменную
- Порты и настройки БД прописаны в docker-compose.yml.
- Включить/отключить Swagger UI можно в application.yml

---

## Развертывание

1. Убедитесь, что Docker и Docker Compose установлены.
2. Установите и откройте проект.
3. В корне проекта выполните:

```bash
  docker-compose up --build
```

4. Приложение будет доступно по адресу: http://localhost:8080/api. 
5. Swagger UI доступен: http://localhost:8080/api/swagger-ui/index.html
